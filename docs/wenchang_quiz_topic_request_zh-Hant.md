# 臺日交流QR問答闖關：題目素材協助說明

製作: 日本石橋小學 松尾 亮佑

## 活動簡介

5月19日的臺日交流活動中，日本石橋小學準備進行一個「校園QR問答闖關」。

日本學生會分組拿著iPad，在校園中尋找QR碼。掃描QR碼後，Safari會打開一道選擇題。學生們會和隊友討論，選出答案。答題後，畫面會顯示答對或答錯，以及簡短解說。

這個活動的目的，是讓日本學生一邊遊戲，一邊更了解臺灣、嘉義，以及嘉義縣文昌國民小學。

## 題目類型

活動中有兩種題目。

- J題: 用日文出題，內容希望和臺灣、嘉義、文昌國小有關。
- C題: 用繁體中文出題，讓日本學生接觸中文並和同伴討論。

這次想請您協助的是J題的素材。J題會由日本方面整理成日文題目。

## 希望提供的資料

不需要寫成完整的題目。只要提供幾個簡單的資訊、關鍵詞或小故事即可。

特別希望是「文昌國小的學生大多都知道」的內容，這樣日本學生能更貼近交流對象的學校生活。

例如:

- 文昌國小的特色
- 文昌國小的校園生活、活動、習慣
- 嘉義的食物、自然、文化、景點
- 臺灣小學生的日常
- 日本學生可能會覺得有趣或驚訝的事情
- 可以改寫成四選一題目的簡單知識

如果方便，請附上正確答案或一兩句說明。

## 範例

- 素材: 文昌國小學生常常參加或重視的活動是什麼？
- 正確答案或說明: ________
- 可以成為題目: 「文昌國小的學生常常參加的活動是什麼？」
