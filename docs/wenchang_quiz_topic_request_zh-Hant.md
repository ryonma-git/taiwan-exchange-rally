# 臺日交流QR問答闖關：題目素材協助簡介

製作: 日本石橋小學 松尾 亮佑

## 簡要說明

5月19日的臺日交流活動中，日本石橋小學準備進行一個使用QR碼的交流問答活動。

日本學生會以小組方式閱讀題目、和同伴討論，並回答選擇題。活動的目的，是讓日本學生透過遊戲式的方式，更了解臺灣、嘉義，以及嘉義縣文昌國民小學。

這份資料只保留與「題目素材提供」有關的簡短說明。活動的詳細進行方式會由日本方面準備。

## 想請教的內容

這次想請您協助的是「用日文出題給日本學生回答」的題目素材。題目會由日本方面整理成日文。

不需要寫成完整的題目。只要提供幾個簡單的資訊、關鍵詞或小故事即可。

特別希望是「文昌國小的學生大多都知道」的內容，這樣日本學生能更貼近交流對象的學校生活。

例如:

- 文昌國小的特色
- 文昌國小的校園生活、活動、習慣
- 嘉義的食物、自然、文化、景點
- 臺灣小學生的日常
- 日本學生可能會覺得有趣或驚訝的事情
- 可以改寫成四選一題目的簡單知識

如果方便，請附上正確答案或一兩句說明。
