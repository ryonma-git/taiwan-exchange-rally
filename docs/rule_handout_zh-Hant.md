# Japan–Taiwan School Discovery Rally 活動規則一覽

## 這是什麼活動？

這是「校園QR問答闖關」。  
每一隊使用一台 iPad，在學校裡尋找QR碼。  
掃描QR碼後，Safari會開啟題目頁面。  
隊員一起討論，再從四個選項中選出答案。

## 活動流程

1. 在開始頁面輸入隊名，按下 Start。
2. 計時器開始，限制時間是 20 分鐘。
3. 在指定區域尋找QR碼。
4. 掃描QR碼，和隊友討論後作答。
5. 時間到時，立刻回到體育館。
6. 開啟 Result 畫面，截圖後用 LoiLoNote 提交。

## 題目種類

- J題：用日文出題，內容是臺灣、嘉義、文昌國民小學。
- C題：用繁體中文出題，內容是日本、大阪、石橋小學。
- J題和C題各20題，總共有40題。
- 20分鐘內很難全部完成，所以選擇哪些QR碼也是策略。

## 最重要的玩法

這個活動不是只看誰跑得快。  
最重要的是「互相詢問、互相教學」。  

- 看不懂日文時，可以問日本隊友。
- 不知道臺灣或日本的事情時，可以問隊友。
- 可以用日文、中文、英文，也可以用手勢和圖片來溝通。

## 分數與作答規則

- 答對題目可以獲得分數。
- 分數會自動累積在 iPad 的頁面中。
- 每一題只能回答一次。
- 答案送出後不能更改。
- 作答前一定要和隊友討論。
- 最後分數最高的隊伍獲勝。

## 翻譯鑰匙

每一隊一開始有 3 把「翻譯鑰匙」。  
使用翻譯鑰匙後，可以看到該題的翻譯。  
使用翻譯鑰匙不會扣分。  

但是鑰匙數量有限，請先問問隊友。  
真的不懂時，再使用翻譯鑰匙。

## 寶箱QR

校園中可能有「寶箱QR」。  
掃描寶箱QR後，可以獲得 1 把翻譯鑰匙。  
寶箱QR不會增加分數，但可以幫助隊伍挑戰難題。

## 安全規則

- 走廊不要奔跑。
- 可以尋找的地方：西校舍、體育館、中庭。
- 操場和東校舍沒有QR碼，請不要進入。
- 不要進入其他班級正在上課的區域。
- 不要撕下、破壞或帶走QR碼。
- 不要大聲說出QR碼的位置或答案。
- 禁止上網搜尋答案。

## 結束時要做什麼？

時間到後，回到體育館。  
打開 Result 畫面，確認隊名、總分、完成題數。  
請將 Result 畫面截圖，交給老師。

## 一句話提醒

和隊友合作，互相幫忙，享受20分鐘的校園QR問答闖關！
